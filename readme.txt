2Checkout integration for WordPress. Collect donation and payments without any E-commerce programs

== Description ==

WP Super Pay is a standalone plugin that helps you collect donation and payments using 2Checkout payment gateway service.

You can create multiple payment forms with different configurations like payment type, amount, currency etc.

See more about about [2Checkout](https://www.2checkout.com/)

== Installation ==

1. Install `WP Super Pay` using the native Plugin installer, or download the zip and extract it in the wp-content/plugins/ directory.
2. Activate the plugin through the `Plugins` list
3. Create Payment Forms
4. Place the shortcode in any post or page.

Please [reach out to us](https://help.codexpert.io/?utm_campaign=wporg) if you need any help.

== Frequently Asked Questions ==

= Does it require WooCommerce? =

No. It's a standalone plugin. Which means you can collect donation or payment withour an e-Commerce program.

= Can I collect recurring payments? =

Yes, you can.

= Can I create multiple payment forms? =

Yes, you can create as many payment forms as you want.

= Where can I see the payments I received? =

There is a separate screen to display the payments you received.
